# Trò chơi rắn săn mồi

*_Đây là trò chơi trên cửa sổ command line_*

- Hướng dẫn chơi
  - Điều khiển bằng các nút w, a, s, d hoặc các nút số bên phải 8, 4, 2, 6
  - Dừng game : space
  - Quay lại : esc
